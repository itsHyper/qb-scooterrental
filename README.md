# Installation
- Download ZIP
- Drag and drop resource into your server files
- Start resource through server.cfg
- Restart your server.
# Dependencies 
- [qb-target](https://github.com/BerkieBb/qb-target)
- [qb-menu](https://github.com/qbcore-framework/qb-menu)
